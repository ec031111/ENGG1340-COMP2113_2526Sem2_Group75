# World Overview: The Realm of Aethoria

Aethoria is a fractured continent where five city-states survived the Collapse of Crowns.
Ancient roads still connect the capitals, but monsters, raiders, and cursed storms make travel dangerous.

The game takes place during the Seventh Defensive Campaign, when the Dark Lord Malachar pushes from the Ashen Frontier.
Every round in the arena represents one strategic phase in this continental war.

Core worldview:
- Hope is scarce but organized resistance is still possible.
- Heroes are not chosen by destiny; they are recruited, trained, and tested.
- Victory is temporary, but survival builds momentum for the next dawn.

These themes explain why the game loop focuses on resource discipline, formation planning, and endurance under pressure.
